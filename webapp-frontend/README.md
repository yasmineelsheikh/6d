# Ares Platform Frontend

Modern React/Next.js frontend for the Ares platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Modern, intuitive UI with dark mode support
- Dataset upload and management
- Interactive visualizations (Environment and Trajectory distributions)
- Episode preview
- Augmentation workflow with Cosmos integration
- Curated dataset view and export

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Plotly.js for visualizations
- Axios for API calls
